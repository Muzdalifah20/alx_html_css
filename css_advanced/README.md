# Smiles School Website

A responsive website showcasing tutorials, testimonials, membership info, and FAQs about smiling.

## Features

- Header with navigation and logo
- Main content with hero section, tutorials, testimonials, membership, and FAQ
- Footer with social media icons

## Usage

Open `index.html` in a browser. Ensure images are in the `images` folder.

## Customization

Feel free to style inside `<article>` and add colors, backgrounds, or borders in CSS.  
Do not change the flexbox layout outside `<article>`.
